Cocoapods如何安装

发表于 2016-09-08

此博文为原创，未经博主同意请勿转载

如过你没有VPN那么你需要设置一下gem源，我们使用的是淘宝的ruby源，如果你有VPN可以忽略这一步骤

1.打开你的终端输入gem sources -l

如果你输入完的结果和上图保持一致的话，那么你不需要更改，注意是https://ruby.taobao.org/ 原来老的教程讲的是http://ruby.taobao.org/ 但是现在淘宝的ruby更新成了HTTPS协议，所以原来的已经不能使用了

2.更新你的源地址
如果你输入gem sources -l之后出现的不是如上图所显示的内容，那么请输入如下命令，比如说你显示的是https://rubygems.org/ 那么输入gem sources --remove https://rubugems.org/ 然后输入gem sources --add https://ruby.taobao.org/ 最后输入gem sources -l如果显示如上图所示表示已经正确了，可以进行安装。

安装CocoaPods在 Mac 终端（Terminal ）上,输入命令行： sudo gem install cocoapods 如果被要求输入Mac的密码，就输入密码，之后等上一个命令执行完毕的时候输入 pod setup --verbose 来完成安装，这个过程可能需要几分钟。 verbose 选项，会让你看到加载的进程。

pod setup这个步骤会持续几分钟，你要做的就是等待。 安装完毕之后可以输入如下命令pod search json

如果出现上图所示代表安装成功~
